# 使用说明

flexScan -h
应该都能看明白

### 目录扫描

默认是一个小字典，其他和dirsearch差不多，大字典(-d b)，指定后缀是大字典才有的，用其他字典没效果。

### 端口扫描

目前只前只能机遇http请求模式的端口扫描。

默认进行判活，再进行web端口扫描，结果输出状态码、title（如果有）。
